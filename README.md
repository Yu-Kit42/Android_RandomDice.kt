# Android_RandomDice.kt
코틀린을 이용한 난수값 생성과 주사위 굴리기

## 기능
### 1 ~ 1000 사이에 40개 이하의 난수를 정렬하여 생성해주는 페이지
### 6개 이하의 주사위 눈을 랜덤하게 보여주는 페이지
생성한 랜덤값을 저장할수 있는페이지 (미완성)
